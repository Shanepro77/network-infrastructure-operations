# Network Infrastructure Operations

This repository demonstrates hands-on experience in designing, configuring, and automating network infrastructure in an operations-focused environment. It emphasizes secure access, proper segmentation, and automation for repeatable deployments.

## Included Labs

### 01-switching-vlans-subnetting
- Configure Cisco switches
- Implement VLANs and subnetting
- Test connectivity and segmentation

### 02-secure-ssh-access-cisco
- Configure secure remote access using SSH
- Apply role-based access control
- Validate authentication and logging

### 03-network-automation-basics
- Automate network tasks with scripts
- Perform configuration backups and restores
- Validate automation and error handling

## Lab Features
- Structured naming conventions
- Verification steps for each configuration
- Operational considerations and troubleshooting notes
